# 🎯 Bullseye Tracking Pan-Tilt System

A real-time computer vision system developed for a dual-servo pan-tilt platform. The camera is mounted on a two-axis mechanism where one servo controls horizontal movement and the other controls vertical movement.

## Features

* Bullseye target detection
* Real-time target tracking
* Kalman Filter based prediction
* Face masking to reduce false detections
* Dual-servo pan-tilt control
* TTL serial communication
* Configurable tracking parameters

## Technologies

* Python
* OpenCV
* NumPy
* PySerial

## Applications

* Robotic vision systems
* Camera tracking platforms
* Autonomous targeting projects
* Computer vision research
